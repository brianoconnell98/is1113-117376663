Cloud 9

Amazon Web Services (AWS) is a part of Amazon that provides on demand cloud computing platforms to individuals, companies and governments, on a paid subscription basis. The technology allows subscribers a full-fledged virtual cluster of computers, available all the time, through the Internet. Amazon's Web Services version of virtual computers have most of the attributes of a real computer including hardware CPU and GPU for processing, local/RAM memory, hard-disk/SSD storage, a choice of operating systems networking and pre-loaded application software such as web servers, databases, etc. Each user is able to access there database on any computer anywhere in the world at anytime.
